# YouTube Search Frontend

YouTube 영상 검색 프론트엔드 애플리케이션 - 직관적인 UI로 YouTube 영상을 검색하고 필터링할 수 있는 웹 애플리케이션입니다.

## ✨ 기능

### 검색 기능
- **키워드 검색**: 검색어 입력으로 YouTube 영상 검색
- **제목 필터링**: 검색어가 제목에 포함된 영상만 표시 (백엔드에서 처리)
- **실시간 검색**: 검색 버튼 클릭으로 즉시 결과 표시

### 필터링 옵션
- **정렬 기준**: 관련도순, 최신순, 조회수순, 평점순, 제목순
- **결과 수**: 10개, 25개, 50개
- **영상 길이**: 전체, 1분 미만 (Shorts), 4분 미만, 4-20분, 20분 이상
- **조회수 범위**: 최소/최대 조회수 입력

### 페이지네이션
- **다음/이전 페이지**: 페이지 토큰을 활용한 페이지 이동
- **페이지 정보 표시**: 현재 페이지 번호, 결과 범위, 총 페이지 수
- **자동 스크롤**: 페이지 이동 시 상단으로 자동 스크롤

### UI/UX
- **반응형 디자인**: 다양한 화면 크기에 대응
- **그라디언트 디자인**: 모던한 UI 디자인
- **영상 카드**: 썸네일, 제목, 채널명, 조회수, 좋아요 수 등 표시
- **로딩 상태**: 검색 중 로딩 표시
- **에러 처리**: 친절한 에러 메시지 표시

## 🛠 기술 스택

- **언어**: TypeScript
- **프레임워크**: React 19.2.0
- **빌드 도구**: Vite 7.2.4
- **라우팅**: React Router DOM 7.11.0
- **스타일링**: CSS3 (그라디언트, 애니메이션)

## 🚀 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 변수 설정 (선택사항)

`.env` 파일을 생성하고 (선택사항):

```
VITE_API_BASE_URL=http://localhost:8080
```

기본값은 `http://localhost:8080`입니다.

### 3. 개발 서버 실행

```bash
npm run dev
```

애플리케이션이 `http://localhost:5173`에서 실행됩니다.

### 4. 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 🎯 주요 기능

### 검색 폼

검색어를 입력하고 다양한 필터 옵션을 선택할 수 있습니다:

- **검색어 입력**: 키워드 입력 필드
- **정렬 기준**: 드롭다운으로 선택
- **결과 수**: 10/25/50개 선택
- **영상 길이**: 전체/Shorts/4분 미만/4-20분/20분 이상
- **조회수 범위**: 최소/최대 조회수 입력

### 검색 결과

- **영상 카드**: 썸네일, 제목, 채널명, 메타 정보 표시
- **결과 정보**: 총 결과 수, 현재 표시 중인 범위
- **YouTube 링크**: 카드 클릭 시 YouTube로 이동

### 페이지네이션

- **이전/다음 버튼**: 페이지 이동
- **페이지 정보**: 현재 페이지 번호, 결과 범위, 총 페이지 수
- **자동 스크롤**: 페이지 이동 시 상단으로 이동

## 🎨 UI 특징

### 디자인
- **그라디언트**: 보라색 그라디언트 테마
- **카드 디자인**: 그림자와 호버 효과가 있는 영상 카드
- **반응형 그리드**: 화면 크기에 따라 자동 조정되는 그리드 레이아웃

### 사용자 경험
- **로딩 상태**: 검색 중 버튼 비활성화 및 로딩 텍스트
- **에러 처리**: 명확한 에러 메시지 표시
- **빈 상태**: 검색 전 안내 메시지
- **숫자 포맷팅**: 조회수, 좋아요 수 등을 K/M 단위로 표시

## 🔗 백엔드 연동

프론트엔드는 `http://localhost:8080`의 백엔드 API와 통신합니다.

Vite의 프록시 설정으로 `/api` 요청이 자동으로 백엔드로 전달됩니다.
